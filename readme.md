## Требования
- Python 3.10 или выше
- pip (устанавливается вместе с Python)

## Установка и запуск

1. Распакуйте архив в любую папку на компьютере.

2. Откройте командную строку  и перейдите в папку проекта:
   cd C:\путь\к\папке\3dplatform

3.Создайте виртуальное окружение (один раз):
python -m venv venv

4. Активируйте виртуальное окружение:
Windows: venv\Scripts\activate

5. Установите библиотеки:
pip install -r requirements.txt

6.Выполните миграции базы данных:
python manage.py migrate

7.Создайте суперпользователя (администратора):
python manage.py createsuperuser

8.Запустите сервер:
python manage.py runserver

9.Откройте браузер и перейдите по адресу:
http://127.0.0.1:8000/
